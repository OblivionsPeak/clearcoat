    How Clearcoat works

    The workflow

    1. Load your car's template PSD (left panel) — Clearcoat extracts the wireframe so you can see where panels are.
    2. Build layers — + Image for logos, + Text for names/numbers, + Fill for shapes & gradients, + Pattern for SimTex textures, + Library for ready-made graphics. Or Import current car paint to start from your Trading Paints livery.
    3. Give every layer a finish — click a layer, pick a material (pearl, candy, flake…), fine-tune it. The Base coat row at the bottom of the layer list is clickable too — that's how the whole car gets a material.
    4. Wand (W) — click a color on the livery to select it (Shift+click = that color everywhere). Give the selection a finish, or check Recolor first to repaint it.
    5. Preview — Shine (L) sweeps a light across the sheet so finishes show; Spec view (S) shows the raw material data.
    6. Save to iRacing — link your paints folder once, keep the sim showroom open, and every save appears on the real car in seconds. Your Trading Paints file is backed up automatically; Restore brings it back.

    Good to know

    • 🔒 Lock a full-sheet layer (imported liveries auto-lock) so clicks pass through it.
    • ⧉ duplicates a layer right from its row.
    • Material only stamps a finish without painting — pearl on existing artwork.
    • Blend (inspector) composites a layer over the paint beneath — Multiply for shading/weathering, Screen for glows.
    • Moving layers snap to the sheet center/edges and region-map lines — hold Alt to move freely.
    • Click the same spot repeatedly to cycle through stacked layers.
    • Studio wraps your livery around 3D geometry under showroom light — the honest way to judge pearl, candy, flake and chrome, since they only read on curvature. It docks beside the canvas, so keep editing while it updates live.
    • Region maps label the sheet's panels (Template panel) — hover shows where you are, Regions overlays the map, and Mirror drops a flipped copy of a layer onto the partner panel. Build your own map with Annotate.
    • Updates arrive with a "New version ready" toast — click Reload to apply. The build you're on shows as the version number top-left.
    • Live save needs Chrome/Edge (Brave: enable File System Access in brave://flags).
